# Project Bach Website

这是Project Bach智能音频处理系统的GitHub Pages网站。

## 功能特性

- 📱 响应式设计，支持各种设备
- 🌙 支持深色/浅色主题切换
- 📊 实时显示音频处理统计信息
- 🔍 查看详细的音频处理结果

## 技术栈

- 纯HTML + CSS + JavaScript
- 响应式布局
- 本地存储主题偏好
- GitHub Actions自动部署

## 自动部署

当代码推送到`gh-pages-website`分支时，GitHub Actions会自动部署网站到GitHub Pages。
